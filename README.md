# VeriDegree — Soulbound Academic Diplomas

VeriDegree is a decentralized application (DApp) for issuing university diplomas as **Soulbound Tokens (SBTs)**. Diplomas (PDFs and metadata) are hosted on **IPFS** via Kubo, ensuring data integrity.

---

## 🚀 Quick Start Guide

### 1. Infrastructure (IPFS & Besu Network)

#### IPFS (Kubo)

Authorize the frontend to interact with your local IPFS node:

```bash
ipfs config --json API.HTTPHeaders.Access-Control-Allow-Origin '["http://localhost:3000"]'
ipfs config --json API.HTTPHeaders.Access-Control-Allow-Methods '["PUT", "POST", "GET"]'
ipfs daemon
```

#### Besu QBFT Network (Docker)

Launch the 4-node private blockchain:

```bash
cd besu-qbft
docker-compose up -d
```

*The network is available at `http://localhost:8545` (Node 1).*

---

### 2. Smart Contracts (Foundry)

Deploy the Soulbound contract to your Besu network:

```bash
cd contracts
forge script script/Deploy.s.sol --rpc-url http://localhost:8545 --broadcast --private-key 8f2a55949038a9610f50fb23b5883af3b4ecb3c3bb792cbcefbd1542c692be63 --legacy
```

*Note: The private key above is a default dev key for the Besu network.*

---

### 3. Administration (Role Management)

To allow a new user to issue diplomas, you must assign them the `MINTER_ROLE`.

#### Create a User

You can generate a new wallet (private key and address) directly using `cast`:

```bash
cast wallet new
```

*Note: Safely store the displayed **private key** so you can import it into MetaMask.*

#### Import into MetaMask

1. Open **MetaMask**.
2. Click the account selector (top center).
3. Click **"Add account or hardware wallet"**, then **"Import account"**.
4. Paste the **private key** generated by `cast`.

#### Grant the MINTER Role

Once you have the address (e.g. `0x123...`), use the administrator account to grant permissions:

```bash
cast send <CONTRACT_ADDRESS> "grantRole(bytes32,address)" 0x9f2df0fed2c77648de5860a4cc508cd0818c85b8b8a1ab4ceeef8d981c8956a6 <NEW_USER_ADDRESS> --rpc-url http://localhost:8545 --private-key 8f2a55949038a9610f50fb23b5883af3b4ecb3c3bb792cbcefbd1542c692be63
```

*Note: The private key `8f2a5...` is the default administrator key.*

---

### 4. Frontend (Next.js)

1. **Configure**: Open `frontend/src/config/contract.ts` and update `CONTRACT_ADDRESS` with the address generated by the deployment script.
2. **Run**:

```bash
cd frontend
npm install
npm run dev
```

3. **Access**: Open **[http://localhost:3000](http://localhost:3000)**.

---

## 🛠 Prerequisites

* Docker & Docker Compose
* Foundry
* Node.js (v18+)
* Kubo (IPFS)
* MetaMask (connect to `http://localhost:8545`, Chain ID `1337`)
